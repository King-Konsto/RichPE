# RichPE
Metadata hash incorporating the Rich Header for robustness against packing and other malware tricks

## RichPE Usage:
```
usage: python3 richpe.py [file_path]
```

## Spoof Check Usage:
```
usage: python3 spoof_check.py [file_path]
```

## Dependencies:
### pefile
```
pip3 install pefile
```
